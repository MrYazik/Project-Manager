# Project Manager - управление вашими проектами
Данный проект поможет организовать информацию о ваших проектах, задачи, схему работы программы 

## Задачи которые решает Project Manager

- Создание задач внутри проекта, разделение задач на типы
- Создание заметок к каждому файлу, которые помогут команде ореинтироваться в проекте
- Отображение работы над проектом в виде статистики (работа над проектом в месяц, количество созданных проектов)
- Встроенный таймер pomodoro для работы над задачей без переработок

## Планируемый интерфейс

1. Главное меню, сверху панель со всеми проектами:
  - Слева -- панель-список, список всех проектов, кнопка создать, когда наводишься на кнопку из списка проекта будет возможность удалить проект (если нажать, будет окно для подтверждения)
  - Когда кнопка будет нажата плавно открывается меню справа, это и будет меню проекта (описывается в следующем меню)
  - Сверху кнопка - скрыть меню всех проектов
  - Если меню проекта не открыто, там отображается надпись
  - Кнопка нстроек

2. Меню проекта
  - Кнопки настройки именно данного проекта (модули).
  - Сверху модуль - README.md, который подтягивается из привязанной папке к проекту, если нету, то из собсвенной папки. В данном модуле будет кнопка изменить, которая открывает редактор
  - Модуль - Идеи, список файлов в которых описанны новые идеи. На файл можно нажать и он откроется в этом модуле. В кадом файле  снизу отображаются задачи, которые соответсвуют данной идее
  - Модуль - Список задач. Каждой задаче привязывается свой файл идеи. Кнопка создания задачи.
  - Размер модулей можно менять
